【ジャンプタイトルプラグイン jumptitle ver1.0.0】

====================================================================================================
■ 概要
====================================================================================================
backTitleを変更するプラグインです。
location.href = "./index.htmlから[jump target=指定の後処理シナリオファイル]に変更します。
※注意！！
当プラグインは、ティラノビルダーの標準関数を上書き（オーバーライド）します。

■ 機能
====================================================================================================
1.ロールボタンのタイトルへ戻るの機能を変更する

====================================================================================================
■ 使い方
====================================================================================================
1.data/others/plugin にjumptitle{バージョン}.zipファイル内の「jumptitle」フォルダをコピーてください。

  ※下記構成で2ファイルがあるか確認してください。
  data/others/plugin/jumptitle/init.ks
  data/others/plugin/jumptitle/jumptitle.js

2.first.ksからプラグインを呼び出してください。

  [plugin name=jumptitle dispose="dispose.ks"]

  disposeパラメータに指定したシナリオファイルが、「タイトルへ戻る」が押された際に呼び出されます。
  省略された場合は、「first.ks」が呼び出されます。

3.後処理シナリオファイル(dispose.ks)について

  タイトルへ戻る際の後処理をこのシナリオファイルに書きます。
  基本的には、イメージやメッセージ枠の消去、音楽や動画の停止を行います。
  そして、最後にタイトルへのジャンプを書いておきます。

  【dispose.ks】

    ; TODO ここに後処理を書く
    ; 以下サンプル

    ; レイヤをクリア
    [freeimage layer="base"]
    [freeimage layer="0"]
    [freeimage layer="1"]
    [freeimage layer="2"]
    [clearfix]

    ; メッセージボックスを非表示
    [layopt layer="message" visible=false]

    ; メニューボタンを非表示
    [hidemenubutton]

    ; 動画を停止
    [stop_bgmovie]

    ; BGMを停止
    [stopbgm]

    ; SEを停止
    [stopse]

    ; タイトルへジャンプ
    [jump storage="title.ks"]
    [s]
